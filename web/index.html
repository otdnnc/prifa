How do I get a token?
  1. Dev mode: click Get dev token if the server runs with -dev-tokens.
  2. CLI: go run ./cmd/token -sub me -secret "$PRIFA_JWT_SECRET"
  3. Your backend: HS256-sign with the same secret (see README).
